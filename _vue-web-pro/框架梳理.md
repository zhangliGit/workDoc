### 项目目录结构

**src目录**

+ src
  + api 存放不同模块的api接口
  + assets 存放项目公共的静态文件
    + js
    + css
    + image
  + components 存放公共组件
    + custorm 自定义符合所有项目使用的组件
    + common 封装当前项目公用组件
    + other 框架封装的组件
  + config
   + defaultSetting.js 项目默认配置项 主题颜色等等
   + router.config.js 路由配置
  + core 核心js库
  + layouts 公共基础布局路由
  + router 路由变量配置
  + store vuex数据状态管理
  + utils js工具库
  + view 项目路由界面
  + App.vue 入口路由
  + main.js 入口文件
  + permisson.js 权限控制
