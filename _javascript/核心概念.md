### js隐式转化

**基本类型转化**

`加减乘除`

+ 字符串加数字时，数字转化为字符串
+ 字符串减数字或者数字减字符串时，字符串转为数字，如果字符串不是纯数字就会转化为NaN
+ 乘除，大于小于跟减一样

```
1 + 'abc' // 1abc
1 - 'abc' // NaN
2 * '12' //23
```
`比较操作`
+ undefined == null
+ 数字和字符串比较，字符串转数字
+ 字符串和字符串比较，字符串转数字
+ 字符串和布尔比较，两种转数字

```
undefined == null // true
1 > '12' // false
undefined > 2 // falee
```

**引用类型转化**

+ 引用类型转数字用valueOf()
+ 引用类型转字符串用toString()
